前言

一个不太纯粹的程序员
断断续续写过几篇文章，由于各种琐事都没有坚持下来。

2014年初我转行了

曾经我是一名机务，如果没有转行，也许某一天你会坐上我修过某些零件的飞机。对于几乎没有计算机编程基础的我，成为一个合格的程序员似乎是一件很艰难的事。虽然家人极力劝阻，但我依然裸辞了。

这几年我从小白开始，一路风风雨雨，陆续担任过几家互联网公司研发总监，再到现在创业，回首过往，收获颇多。记忆抵不过流年，因此还是想记录一下自己的心得经历，愿与所有正在努力的朋友一起分享，共勉！

菜鸟腾飞之路

学编程，如段誉学六脉神剑，刚开始似懂非懂，然后渐入佳境，最后炉火纯青。曾经每晚做梦写代码，是最难过的记忆之一。

任何一个大牛，都是从小白起步的。
不是科班，没计算机基础
英语太差，看不懂
前途迷茫，不知道学什么
害怕学不会，一看代码就懵逼
被其他同事鄙视

我也曾经是个一无所知的菜鸟，我相信很多入行的朋友，都有过以上的某些心理。如果现在还有上面的想法,我以自己真实经历告诉你，这些都不是问题。

我非计算机专业，英语勉强过了4级，没上过培训班，甚至很多人都和我一样，在学校的时候，虽然学了C语言，但是却连句“hello world”都写不出来。但是我坚持下来了，走到了今天可以说，只要不是文盲，会写英文字母，就一定能行。初入行，能力弱，被人鄙视很正常，毕竟整个行业都存在鄙视链呢，何必太在意呢。端正心态，走自己的路。

如果不知道自己该学什么，该如何入门，那希望你不要放弃，在以后的文章里，我会分享自己的成长历程，拨开云雾见天日 守得云开见月明。

万事开头难，从菜鸟到大神，要做的，是坚持！

人生苦短，该学什么

自从Python大热之后，就带火一句话，“Life is short，you need python”。
但在这个信息技术飞速变化的时代，光用一门语言显然不足以撑起自己爆栈的理想，当然了也不太可能样样技能精通。我的建议是从精通一门语言开始，向其他语言、技能辐射，学的再多如果不用，慢慢也都忘记了。不用担心自己的技能栈太少，只要肯学习，随着工作经验的加深，积累的越来越多，自然能力就强了。
所以呢，人生苦短，赶快入坑，需要什么学什么吧。
