vim基本操作+常用命令

发表于 2018-11-03 | 更新于 2018-12-26 | 分类于 杂谈 | 评论数：
本文字数： 2.4k | 阅读时长 ≈ 2 分钟

本文分基础篇、技巧篇和进阶篇三个部分，包含了大部分常用命令，用于帮助新手快速入门。适合刚接触vi的人翻阅，或者作为日常手册查阅，但不包含VIM所有的操作。所有操作均基于Linux

基础篇

vi、vim

  • 在一般发行的Linux系统中，几乎都默认安装了vi编辑器，具有基本的编辑功能，有必要掌握它的基本用法
  • vim则是vi编辑器的高级版，具有相对多一些的操作
  • 如果本文以下某些操作在你的vi中无法正常使用，那你就应该安装vim了。安装过程就不多说，自行使用apt-get、yum 等方式安装

工作模式

先来使用vim打开一个文件试试看，filename可以是已存在的文件，也可以不存在

此时可以看到打开的窗口中有光标闪烁，是不是就可以开始编辑了呢？随便按几个按键试试看，汗…发现不是我们想要的结果。这里就引出了vim的操作模式，模式名称请勿纠结，理解含义即可

  • 普通模式：也就是刚打开文件的时候，此时可以移动光标，切换模式，按ESC回到该模式
  • 编辑模式：顾名思义，该模式下可以进行编辑，按i,a等进入该模式
  • 选择模式：可以选择某段文字，某个区块，按v，C-v进入该模式
  • 命令模式：可以像编辑器发送一些指令，例如：保存、退出、显示行号等操作，按:进入该模式

所有模式都要从普通模式进入，在其他模式下使用ESC键，即可回到普通模式

基本操作

  1. 当前情况处于普通模式，可进行光标移动操作:
  • h 左
  • j 下
  • k 上
  • l 右
  1. 使用以下任意命令进入编辑模式:
  • i 从光标当前位置开始插入
  • a 从光标当前位置后一位开始插入
  • o 从光标当前位置下一行开始插入
    此时想要移动光标，则需要退出编辑模式，按ESC键即可
  1. 保存内容,如果当前在普通模式下，按:进入命令模式,可进行保存、退出等操作
  • e 刷新内容
  • w 保存
  • wq 保存并退出
  • q! 强制退出

至此，已经可以顺利的编辑并保存一个文件了，但这还不远远不够

技巧篇

光标移动

假如仅仅使用h,j,k,l进行光标移动，那是不可忍受的事情，光标一次只能移动一位。下面的操作可以极大提高效率:

  • w 光标移动到下一个单词的开头
  • e 光标移动到下一个单词的结尾
  • 0 光标移动到当前行首
  • ^ 光标移动到当前行首，非空白位置
  • $ 光标移动到当前行尾
  • g_ 光标移动到当前行尾，非空白位置
  • gg 光标移动到首行
  • G 光标移动到末行

可以尝试将命令换成大写，看看有什么效果哦

选择复制粘贴

在vim中，删除操作基本都属于剪切（非系统剪贴板）

  • x 删除光标所在（或者选中）的字符
  • dd 删除光标所在行
  • yy 复制光标所在行
  • p 将粘贴板内容粘贴到光标后

选择复制：

  1. 进入选择模式：
  • v 字符复制
  • C-v 区块复制
  • Shift-v 行复制
  1. 移动光标进行选择
  2. 复制粘贴
  • y 复制选中内容
  • p 粘贴

使用数字前缀，进行多字符，多行操作，例如:

  • 2dd 删除两行
  • 2yy 复制两行
  • 10j 光标向下移动10行
  • 30gg 光标跳转到第30行

回退、反回退

  • u 回退
  • C-r 反回退 ，这个是ctrl+r 组合键

显示行号

命令模式下执行:

  • set nu 显示行号
  • set nonu 关闭行号

分屏

命令模式下执行：

  • sp 水平分屏
  • vsp 垂直分屏
  • C-w 分屏切换
  • C-l 切换右边屏
  • C-h 切换左边屏

高级篇

这里所谓高级篇，其实就是不常用，但是关键时刻很好用的操作

搜索

命令模式下输入/xxx（或?xxx），按下enter，光标将跳转到搜索到的第一个xxx字符串位置,按n查找下一个

  • / 是向下搜索
  • ? 是向上搜索

替换

  • 命令模式下，输入s/abc/xyz ，按下enter，将使用xyz替换光标所在行的abc
  • 命令模式下，输入1,$s/abc/xyz/g ，按下enter，将使用xyz替换文本中所有的abc

其他更多替换方式，可以自行研究

区块操作

批量在行首插入字符，在多行注释的时候十分有用，下面以多行注释操作为例

  1. 普通模式下，按下0，光标移至行首

  2. 按下C-v，上下移动光标，发现可以垂直选中

  3. 按下I（注意是大写），进入插入模式，假设输入一个#

  4. 按下ESC，发现之前被垂直选中的多行首位均已加上#

不退出vim，执行Linux系统命令

在命令模式下，输入!后面加系统命令，即可执行，例如
!ls 列出当前目录文件，其他命令类似，以!开头

宏的简单使用

宏可以简单理解为一系列操作的快捷键,这里以一个实例说明

  • 需求：批量在100行文字首位，加上分号”;”

  • 分析：光标移至行首 –> 进入编辑模式 –> 插入; –> 退出编辑模式 –> 光标移至下一行 –> 循环以上操作

  • 实现：

  1. 普通模式下，按qa(也可以是qb、qc… ;a、b、c… 表示寄存器，存放宏)，此时开始进行宏录制

  2. 按0光标移至行首，按i进入插入模式，键入; ,按esc退出编辑模式，按j光标移至下一行

  3. 按q退出宏录制。此时宏录制完成，存放在寄存器a中

  4. 普通模式下，输入@a，此时发现行首成功加上分号,且光标跳转到了下一行,说明宏运行成功。如果需要操作100行，则输入100@a

  5. 命令模式下输入reg a,可以查看寄存器a的宏内容

  6. 修改宏内容,命令模式下输入let @a=' ,然后按下ctrl-r ctrl-r a ,将自动补全寄存器a中的宏内容，然后进行编辑，补全最后的单引号,即可修改成功。

结束语

以上操作是vim常用的操作，用来完成一般的编辑已经足够。如果想让你的vim具有更强大的超能力，就需要学习更多的光标移动、插件、配置、宏等其他知识。
